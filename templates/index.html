Sentiment Analysis

Data Science is increasingly becoming important, the answer lies in the fact that the value of data is soaring heights. It explains how digital data is transforming businesses and helping them make sharper and critical decisions.


For Educational Purpose Only

Analyse the trends based on the search keyword and plot the sentiment with data extracted from various sources like Twitter, Finviz for Stocks and also from Global News articles. Search Ex: TSLA, Crypto, IPL


Three different charts will be displayed after the search, more details below about each chart. The Sentiment is calculated as POSITIVE or NEGATIVE for each search keyword, ranging from +1(Extreme Positive) to -1(Extreme Negative)


Try Searching Here !!

Twitter

Analyze the sentiment from Twitter for the search key. Each dot on the chart represents a tweet. Limiting to only 500 tweets per search to avoid the API limit burn out.
Ex: Tesla, IPL, MachineLearning

Worldwide News

Analyze the News articles from all over the world using NewsData.io API for the search key. Limited to only 10 articles per search.

Ex: Tesla, Bitcoin, IPL

USA Stocks

Analyze the Stock news and plot the news trends in the Chart. Each bar on the chart represents the sentiment for the day. Only USA[NYSE/Nasdaq] listed Stocks are analysed.
Ex: TSLA, GOOG, MSFT, FB



This application is hosted on AWS.
